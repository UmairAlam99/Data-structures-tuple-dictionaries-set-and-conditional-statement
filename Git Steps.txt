Git Steps:

1) Download and install git from git-scm.com
2) Right Click to the folder and goto git bash terminal
3) To convert folder into repository use : 
	$git init
4) Inorder to check status of git use :
	$git status
5) To change untrack files into stage position use OR convert modified files into stage use:
	$git add -A
6) To configure git globally on your computer while getting message of who are you in terminal use:
	a) $git config --global user.name "<NAME>"
	b) $git config -global user.email "<EMAIL>"
7) To commit/save changes use:
	$ $git commit -m "<MESSAGE>"
8) Create account on github.com
9) Create repo on github 
10) Copy link of github repo
11) Inorder to connect local repo with remote use:
	$git remote add origin <LINK>
12) To push your code to github use:
	$git push -u origin master

13) To clone code to existing repo use:
	$git clone <LINK>
